### Prerequisites


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/baliksjosay/javascript-kampala-site.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start Project 
   ```sh
   npm start
   ```
4. Buld Project
   ```sh
   npm build
   ```
